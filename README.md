# Smart Inventory Flow for WooCommerce

Custom WooCommerce inventory system designed for Kuwait businesses.

## Overview
This project demonstrates a smart inventory synchronization flow where products can be sold as:

- Individual Pieces
- Full Cartons

while sharing the same inventory stock automatically.

---

## Features
- Piece & Carton synchronization
- Shared inventory logic
- Automatic stock deduction
- Real-time inventory updates
- WooCommerce + PHP customization
- Inventory accuracy improvements

---

## Example Logic

24 Pieces = 1 Carton

If a customer orders:
- 1 Carton → system deducts 24 pieces automatically
- 2 Pieces → stock updates instantly
